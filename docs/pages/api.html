API Routes

For Users, there are 4 API routes for the GET request methods.

  • /api/:type
  • /api/:type/:id
  • /api/search/:type
  • /api/:type/limit/:sequence

And for System Owners like Stores or Websites, there are 4 API routes for the GET request methods, and 1 route for each request method of the following, POST, PUT DELETE.

  • For PUT and DELETE methods:
    • /api/:type/:id
  • For POST method:
    • /api/:type/
